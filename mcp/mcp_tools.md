# Org-roam-semantic MCP Server Tools

The org-roam-semantic MCP server provides the following tools:

- **org_roam_semantic_semantic_search**: Semantic vector search using AI embeddings (query, limit, cutoff)
- **org_roam_semantic_contextual_search**: Enhanced search with full context for RAG (query, limit)
- **org_roam_semantic_search_notes**: Basic search across note titles and content (query)
- **org_roam_semantic_create_note**: Create new org-roam notes with structure (title, content, type, confidence, url, metadata)
- **org_roam_semantic_add_daily_entry**: Add structured entries to daily notes (timestamp, title, points, next_steps, tags, entry_type)
- **org_roam_semantic_get_daily_content**: Retrieve daily note content (date optional)
- **org_roam_semantic_generate_embeddings**: Generate semantic embeddings for notes (force optional)
